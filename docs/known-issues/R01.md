## Severity

Critical (historical).

## Summary

When Micrometer was present, `MeteredPlatformTracing` was registered as the primary
`TraceOperations` bean. Because it only overrode a subset of the wide `TraceOperations` API,
relation-aware and link-aware calls fell back to interface default methods on the decorator
instance. This silently degraded ROOT/DETACHED/links semantics.

## Durable fix (Slice 2 + 6)

- Narrow public facade: `traceContext()` + `manual()` only; no behavioral interface defaults on
  span creation paths.
- Single internal boundary: `TracingImplementation.startSpan(SpanSpec)`.
- Metering decorator: `MeteredTracingImplementation` wraps `TracingImplementation` fully (not
  `TraceOperations`).
- Topology proof: `MeteredTopologyMatrixTest`.
